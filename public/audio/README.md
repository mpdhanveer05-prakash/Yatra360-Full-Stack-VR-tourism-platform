# Yatra360 Ambient Audio

Drop CC0-licensed looping ambient tracks here. The audio engine
(`src/lib/ambientAudio.ts`) loads them lazily and silently no-ops if missing.

## Required files

| Filename                   | Used for category | Suggested mood          |
|----------------------------|-------------------|--------------------------|
| `ambient-heritage.mp3`     | `heritage`        | Stone hall reverb        |
| `ambient-temple.mp3`       | `temple`          | Bells, distant chants    |
| `ambient-fort.mp3`         | `fort`            | Desert wind, crow caws   |
| `ambient-museum.mp3`       | `museum`          | Quiet hush               |
| `ambient-nature.mp3`       | `nature`          | Forest birds, leaves     |
| `ambient-spiritual.mp3`    | `spiritual`       | Drone, mantra hum        |
| `ambient-hills.mp3`        | `hill-station`    | Mountain breeze          |

## Source suggestions

- [freesound.org](https://freesound.org) — filter by CC0.
- [Pixabay Audio](https://pixabay.com/music/) — royalty-free loops.

Target: 60–120s seamless loops, 128–192 kbps mp3, normalized to ~ -18 LUFS.
